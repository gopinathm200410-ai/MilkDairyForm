 
🥛 Milk Diary Management System
📖 Description:

The Milk Diary Management System is a Python-based mini project designed to manage dairy products such as Milk, Curd, and Buttermilk. It helps users maintain product stock, set prices, process sales, generate bills, and store sales records. The system provides a simple menu-driven interface for efficient inventory and sales management.

✨ Key Features:

Set product prices
Import and update stock
View inventory details
Sell products and generate bills
Automatic stock deduction
Calculate inventory value
Save sales reports to a file
User-friendly menu system
